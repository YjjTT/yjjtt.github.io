PATH

  1. PATH的作用就是每次在Bash里面输入一个命令时, Bash都会去PATH列表里面去找对应的文件,找到了就执行, 脚本其实就是一个可执行的文件(ls, cd, mkdir..)
  2. 可以输入命令echo $PATH查看所有的路径,Bash就会在这些路径中依次查找
  3. type demo可以查看查找过程
  4. which demo可以看到查找结果
